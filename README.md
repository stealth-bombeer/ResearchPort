## üõ†Project Setup


 Clone the repository
```
https://github.com/stealth-bombeer/ResearchPort.git

```
To start the frontend user interface
```
cd client
npm install
npm dev
```
To start the Node server
```
cd server
npm install
nodemon server
```
To start the Flask Server
```
cd python_backend
pip install
python main.py
```
